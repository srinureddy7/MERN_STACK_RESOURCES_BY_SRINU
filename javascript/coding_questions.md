# JavaScript Array Coding Questions

This repository contains solutions to various JavaScript array coding questions. These are useful for practice and preparation for job interviews.

## Questions and Solutions

### 1. Reverse an Array

```javascript
function reverseArray(arr) {
  return arr.reverse();
}

// Example
console.log(reverseArray([1, 2, 3, 4, 5])); // [5, 4, 3, 2, 1]
```
